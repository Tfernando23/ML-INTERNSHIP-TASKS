**Machine Learning Internship Program**

**Introduction**
This project consists of four machine learning tasks that demonstrate the application of Artificial Intelligence across multiple domains, including classification, natural language processing, deep learning, and recommendation systems. Each task follows a structured workflow that begins with understanding the problem, followed by data preprocessing, model building, and performance evaluation. The objective of this project is to gain practical knowledge of how machine learning models are developed and applied to solve real-world problems efficiently.

**Task 1: Decision Tree (Classification)**

The first task focuses on building a Decision Tree model for classification using the Iris dataset. The process begins with loading the dataset, which contains labeled data representing different flower species. The dataset is then divided into training and testing sets to ensure proper evaluation. A Decision Tree algorithm is chosen due to its simplicity and interpretability. The model is trained by recursively splitting the dataset based on feature values, forming a tree-like structure. After training, the model predicts class labels for the test data. The performance is evaluated using accuracy and classification metrics. Finally, the decision tree is visualized to clearly understand how the model makes decisions at each step.

**Task 2: Sentiment Analysis (NLP)**

The second task involves performing sentiment analysis on textual data. The process begins with creating a dataset of customer reviews along with their sentiment labels. Since machine learning models cannot directly process text, preprocessing is performed, including cleaning and preparing the text data. The text is then converted into numerical form using TF-IDF vectorization, which captures the importance of words. A Logistic Regression model is trained on this transformed data to learn patterns in the text. The model is then used to predict whether new reviews are positive or negative. The performance is evaluated using metrics such as accuracy, precision, recall, and F1-score, providing a detailed understanding of the model’s effectiveness.

**Task 3: CNN (Image Classification)**

The third task focuses on image classification using a Convolutional Neural Network (CNN). The process begins with loading the MNIST dataset, which contains images of handwritten digits. The images are preprocessed by normalizing pixel values and reshaping them to match the input requirements of the CNN model. The model is designed using convolutional layers to extract features, pooling layers to reduce dimensionality, and dense layers for classification. The model is trained over multiple epochs, allowing it to learn patterns from the images. After training, the model is evaluated on test data to measure its accuracy. This task demonstrates how deep learning models can automatically learn and extract features from image data.

**Task 4: Recommendation System (Collaborative Filtering with SVD)**

The final task involves building a recommendation system using collaborative filtering and matrix factorization techniques. The process begins with creating a dataset consisting of users, items, and their corresponding ratings. This data is structured into a format suitable for training using the Surprise library. The dataset is then split into training and testing sets to ensure proper evaluation. The Singular Value Decomposition (SVD) algorithm is used to learn latent features representing user preferences and item characteristics. After training, the model predicts ratings for unseen user-item pairs. The performance is evaluated using Root Mean Squared Error (RMSE), which measures the accuracy of predictions. Finally, the system generates personalized recommendations by predicting ratings for specific users and identifying items with the highest predicted scores. This task highlights how machine learning can be used to deliver personalized user experiences in real-world applications.
